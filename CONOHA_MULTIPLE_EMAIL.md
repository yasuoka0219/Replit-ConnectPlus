# ConoHaで複数のメールアドレスを作成する方法

## ✅ 概要

同じドメイン（`bizcraft-studio.com`）から、複数のメールアドレスを作成できます。

**例:**
- `katsuhiro.okazaki@bizcraft-studio.com`（既に作成済み）
- `tanaka@bizcraft-studio.com`（新規作成）
- `info@bizcraft-studio.com`（新規作成）
- など、必要な数だけ作成可能

---

## 🚀 追加のメールアドレスを作成する手順

### ステップ1: ConoHaコントロールパネルにアクセス

1. **ConoHaコントロールパネルにログイン**
   - https://www.conoha.jp/ にアクセス
   - コントロールパネルにログイン

2. **WINGサービスに切り替え**
   - 上部のサービスアイコンから「WING」をクリック

### ステップ2: メールボックスを作成

1. **「メール」→「メールボックス」を選択**
   - 左サイドバーから「メール」をクリック
   - 「メールボックス」を選択

2. **「+ メールアドレス」ボタンをクリック**
   - 右上の「+ メールアドレス」ボタンをクリック
   - または、「メールボックス作成」ボタンをクリック

3. **メールアドレスを設定**
   - **ドメイン**: `bizcraft-studio.com`（既に取得しているドメインを選択）
   - **メールアドレス**: 新しい社員のメールアドレスを入力
     - 例: `tanaka@bizcraft-studio.com`
     - 例: `yamada@bizcraft-studio.com`
     - 例: `info@bizcraft-studio.com`
   - **パスワード**: 強力なパスワードを設定
   - **容量**: 必要に応じて設定（例: 200MB、500MB、1GBなど）

4. **「作成」をクリック**
   - メールアドレスが作成されます
   - 作成が完了すると、メールアドレス一覧に表示されます

### ステップ3: 作成完了の確認

1. **メールアドレス一覧を確認**
   - 作成したメールアドレスが一覧に表示されます
   - 例:
     - `katsuhiro.okazaki@bizcraft-studio.com`（既存）
     - `tanaka@bizcraft-studio.com`（新規作成）

2. **各メールアドレスの設定を確認**
   - メールアドレスをクリックすると、詳細情報が表示されます
   - Webメールへのアクセス、設定ファイルのダウンロードなどが可能です

## 💡 メールアドレスの命名規則

### 推奨される命名方法

1. **個人名を使用**
   - `tanaka@bizcraft-studio.com`
   - `yamada@bizcraft-studio.com`
   - `suzuki@bizcraft-studio.com`

2. **フルネームを使用**
   - `tanaka.taro@bizcraft-studio.com`
   - `yamada.hanako@bizcraft-studio.com`

3. **役職や部門を使用**
   - `manager@bizcraft-studio.com`
   - `sales@bizcraft-studio.com`
   - `support@bizcraft-studio.com`

4. **組み合わせ**
   - `tanaka.sales@bizcraft-studio.com`
   - `yamada.support@bizcraft-studio.com`

### 注意事項

- **使用可能な文字**: 英数字、ハイフン（-）、アンダースコア（_）、ドット（.）
- **大文字小文字**: 区別されません（`Tanaka@bizcraft-studio.com` と `tanaka@bizcraft-studio.com` は同じ）
- **長さ**: メールアドレスの長さに制限があります（通常、ローカル部分は64文字まで）

## 🔄 システムでの使い分け

### 複数のメールアドレスを用途別に使用

システムでは、環境変数で`SMTP_FROM_EMAIL`を変更するだけで、送信元メールアドレスを変更できます。

**例:**

1. **2段階認証用**
   ```bash
   SMTP_FROM_EMAIL=noreply@bizcraft-studio.com
   ```

2. **顧客へのメール送信用**
   ```bash
   SMTP_FROM_EMAIL=info@bizcraft-studio.com
   ```

3. **サポート用**
   ```bash
   SMTP_FROM_EMAIL=support@bizcraft-studio.com
   ```

### 複数のメールアドレスを同時に使用する場合

システムのコードを変更することで、用途に応じて異なるメールアドレスから送信することも可能です。

現在の実装では、環境変数で1つのメールアドレスを設定しますが、将来的に拡張することもできます。

## 💰 料金について

### メールボックスの料金

- **追加料金**: メールボックスを追加するたびに、追加料金がかかる場合があります
- **容量**: メールボックスの容量によって料金が異なる場合があります
- **詳細**: ConoHaの料金ページで確認してください

### 推奨事項

- **必要な数だけ作成**: 必要な社員分だけメールアドレスを作成
- **共有メールアドレス**: 部門用のメールアドレス（`info@`、`support@`など）も作成可能
- **容量の設定**: 用途に応じて適切な容量を設定

## 🔍 よくある質問

### Q: 何個までメールアドレスを作成できますか？

A: ConoHa WINGのプランによって異なります。プランの詳細を確認してください。

### Q: 既存のメールアドレスを削除できますか？

A: はい、可能です。メールアドレス一覧から削除アイコンをクリックして削除できます。

### Q: メールアドレスのパスワードを変更できますか？

A: はい、可能です。メールアドレスの詳細画面からパスワードを変更できます。

### Q: 複数のメールアドレスを同じメールクライアントで管理できますか？

A: はい、可能です。メールクライアントに複数のアカウントを追加できます。
