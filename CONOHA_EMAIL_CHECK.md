# ConoHaメールアドレスのメール確認方法

## ✅ メールを確認する方法

ConoHaで作成したメールアドレス（`katsuhiro.okazaki@bizcraft-studio.com`）に届くメールは、以下の方法で確認できます。

---

## 🌐 方法1: Webメールで確認（最も簡単）

### ステップ1: ConoHaコントロールパネルにアクセス

1. **ConoHaコントロールパネルにログイン**
   - https://www.conoha.jp/ にアクセス
   - コントロールパネルにログイン

2. **WINGサービスに切り替え**
   - 上部のサービスアイコンから「WING」をクリック

### ステップ2: Webメールにアクセス

1. **「メール」→「メールボックス」を選択**
   - 左サイドバーから「メール」をクリック
   - 「メールボックス」を選択

2. **メールアドレスを選択**
   - メールアドレス一覧から `katsuhiro.okazaki@bizcraft-studio.com` をクリック

3. **「Webメール」をクリック**
   - メールアドレスの詳細画面で「Webメール」ボタンまたはリンクをクリック
   - または、メールアドレス一覧に「Webメール」のリンクがある場合があります

4. **Webメールにログイン**
   - メールアドレス: `katsuhiro.okazaki@bizcraft-studio.com`
   - パスワード: メールボックス作成時に設定したパスワード
   - ログイン

5. **メールを確認**
   - Webメール画面で受信トレイを確認
   - システムから送信された認証コードなどのメールが表示されます

---

## 📧 方法2: メールクライアントで確認（推奨）

メールクライアント（Outlook、Thunderbird、メールアプリなど）を設定することで、パソコンやスマートフォンでメールを確認できます。

### ステップ1: メールクライアントの設定情報を確認

ConoHa WINGのメールサーバー設定情報：

**受信メール（IMAP）設定:**
```
IMAPサーバー: imap.wing.conoha.jp
または
IMAPサーバー: imap.conoha.jp
IMAPポート: 993（SSL/TLS）または 143（STARTTLS）
ユーザー名: katsuhiro.okazaki@bizcraft-studio.com
パスワード: メールボックス作成時に設定したパスワード
```

**受信メール（POP3）設定:**
```
POP3サーバー: pop.wing.conoha.jp
または
POP3サーバー: pop.conoha.jp
POP3ポート: 995（SSL/TLS）または 110（STARTTLS）
ユーザー名: katsuhiro.okazaki@bizcraft-studio.com
パスワード: メールボックス作成時に設定したパスワード
```

**送信メール（SMTP）設定:**
```
SMTPサーバー: smtp.wing.conoha.jp
または
SMTPサーバー: smtp.conoha.jp
SMTPポート: 587（STARTTLS）または 465（SSL/TLS）
ユーザー名: katsuhiro.okazaki@bizcraft-studio.com
パスワード: メールボックス作成時に設定したパスワード
認証: 必要
```

**注意:**
- 正確なサーバー名は、ConoHa WINGのメール設定画面で確認してください
- メールボックスの設定画面に「設定ファイル」のダウンロードリンクがある場合があります
- 設定ファイルをダウンロードすると、メールクライアントの設定が簡単になります

### ステップ2: メールクライアントを設定

#### Outlookの場合

1. **Outlookを開く**
   - 「ファイル」→「アカウント設定」→「アカウント設定」

2. **新しいアカウントを追加**
   - 「新規」をクリック
   - 「手動設定または追加のサーバーの種類」を選択
   - 「次へ」をクリック

3. **アカウント情報を入力**
   - メールアドレス: `katsuhiro.okazaki@bizcraft-studio.com`
   - パスワード: メールボックス作成時に設定したパスワード
   - 「サーバー設定を手動で構成する」にチェック
   - 「次へ」をクリック

4. **サーバー設定を入力**
   - アカウントの種類: IMAP または POP3
   - 受信メールサーバー: `imap.wing.conoha.jp` または `pop.wing.conoha.jp`
   - 送信メールサーバー: `smtp.wing.conoha.jp`
   - ユーザー名: `katsuhiro.okazaki@bizcraft-studio.com`
   - パスワード: メールボックス作成時に設定したパスワード
   - 「詳細設定」をクリック

5. **詳細設定**
   - 受信サーバー:
     - ポート: 993（IMAP SSL）または 143（IMAP STARTTLS）
     - 暗号化: SSL/TLS または STARTTLS
   - 送信サーバー:
     - ポート: 587（STARTTLS）または 465（SSL/TLS）
     - 暗号化: STARTTLS または SSL/TLS
     - 「送信サーバーは認証が必要」にチェック
   - 「OK」をクリック

6. **設定を完了**
   - 「次へ」をクリック
   - 接続テストが実行されます
   - 成功したら「完了」をクリック

#### iPhone/iPadの場合

1. **設定アプリを開く**
   - 「設定」→「メール」→「アカウント」→「アカウントを追加」

2. **「その他」を選択**
   - 「その他」→「メールアカウントを追加」

3. **アカウント情報を入力**
   - 名前: 任意の名前（例: CONNECT+ CRM）
   - メール: `katsuhiro.okazaki@bizcraft-studio.com`
   - パスワード: メールボックス作成時に設定したパスワード
   - 説明: 任意の説明（例: 会社メール）

4. **「次へ」をクリック**

5. **サーバー設定を入力**
   - 受信メールサーバー:
     - ホスト名: `imap.wing.conoha.jp` または `pop.wing.conoha.jp`
     - ユーザー名: `katsuhiro.okazaki@bizcraft-studio.com`
     - パスワード: メールボックス作成時に設定したパスワード
   - 送信メールサーバー:
     - ホスト名: `smtp.wing.conoha.jp`
     - ユーザー名: `katsuhiro.okazaki@bizcraft-studio.com`
     - パスワード: メールボックス作成時に設定したパスワード

6. **「次へ」をクリック**
   - 設定が保存されます

#### Androidの場合

1. **メールアプリを開く**
   - 「設定」→「アカウント」→「アカウントを追加」

2. **「その他」を選択**
   - 「その他」→「メールアカウント」

3. **アカウント情報を入力**
   - メールアドレス: `katsuhiro.okazaki@bizcraft-studio.com`
   - パスワード: メールボックス作成時に設定したパスワード
   - 「手動設定」を選択

4. **サーバー設定を入力**
   - 受信サーバー:
     - サーバー: `imap.wing.conoha.jp` または `pop.wing.conoha.jp`
     - ポート: 993（IMAP SSL）または 143（IMAP STARTTLS）
     - セキュリティ: SSL/TLS または STARTTLS
   - 送信サーバー:
     - サーバー: `smtp.wing.conoha.jp`
     - ポート: 587（STARTTLS）または 465（SSL/TLS）
     - セキュリティ: STARTTLS または SSL/TLS

5. **「次へ」をクリック**
   - 設定が保存されます

## 🔍 メールが届かない場合

### 確認事項

1. **スパムフォルダを確認**
   - メールがスパムフォルダに入っている可能性があります

2. **メールボックスの容量を確認**
   - ConoHaコントロールパネル → 「メール」→「メールボックス」
   - 容量が満杯になっていないか確認

3. **システムのログを確認**
   - Railwayダッシュボード → 「HTTP Logs」
   - `[2FA Email]`で始まるログを確認
   - メール送信が成功しているか確認

4. **環境変数を確認**
   - Railwayダッシュボード → 「Variables」
   - SMTP設定が正しいか確認

## 💡 補足情報

### WebメールのURL

ConoHa WINGのWebメールは、通常以下のようなURLでアクセスできます：

- `https://webmail.wing.conoha.jp`
- または、ConoHaコントロールパネルから直接アクセス

### メールクライアントの推奨設定

- **IMAP**: 推奨（メールがサーバーに残る）
- **POP3**: メールをダウンロードしてサーバーから削除する場合

### セキュリティ

- **SSL/TLS**: 推奨（ポート993、995、465）
- **STARTTLS**: 代替（ポート143、587）
