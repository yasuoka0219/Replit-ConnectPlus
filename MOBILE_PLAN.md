# モバイル対応実装プラン

## 📱 現状の問題点

1. **サイドバー**: 常に表示されており、モバイルでは画面を占有
2. **フィルタフォーム**: 横並びで、モバイルでは見づらい
3. **テーブル**: 横スクロールが必要になる可能性
4. **ハンバーガーメニュー**: 存在しない
5. **ボタンサイズ**: タッチフレンドリーでない可能性

## 🎯 実装内容

### 1. ハンバーガーメニューの追加
- モバイルでサイドバーを非表示
- ハンバーガーアイコンでメニューを開閉
- オーバーレイ表示

### 2. サイドバーのモバイル対応
- デスクトップ: 常に表示
- モバイル: オーバーレイ表示、背景をタップで閉じる

### 3. フィルタフォームのレスポンシブ対応
- モバイル: 縦並び
- タブレット以上: 横並び

### 4. テーブルのモバイル対応
- 横スクロール対応
- またはカード表示に切り替え（オプション）

### 5. ボタン・入力フィールドのタッチフレンドリー化
- 最小タップサイズ: 44x44px
- フォントサイズの調整

### 6. ダッシュボードのレスポンシブ対応
- グリッドレイアウトの調整
- モバイル: 1列
- タブレット: 2列
- デスクトップ: 4列
